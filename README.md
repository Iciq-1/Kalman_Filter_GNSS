# Kalman_Filter_GNSS
This project was done for Bayesian Filtering and Smothing. It tackles with Kalman filtering used on non-linear pseudoranges from satellites.

This code was created as part of a university project. 

## Acknowledgments
The logic and matrix models for the Constant Velocity EKF were heavily inspired by the following paper:
*Wickert, M., & Siddappa, C. (2018). Exploring the extended Kalman filter for GPS positioning using simulated user and satellite track data. In Proceedings of the 17th Python in Science Conference (SciPy 2018).*
